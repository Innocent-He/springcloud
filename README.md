##项目简介
---

该项目是跟着尚硅谷周阳的Springcloud2020做出来的，里面涉及传统的 eureka、hystrix、ribbon，以及最新的 alibaba的 Nacos和sentinel、Seata
不过并没有跟着学习Zookeeper与Consul这两个注册中心
